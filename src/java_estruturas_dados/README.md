# Projeto Java — Estruturas de Dados

Este projeto contém implementações próprias das seguintes estruturas de dados:

- Pilha (Stack)
- Fila FIFO (Queue)
- Lista Encadeada (LinkedList)
- Mapa Hash (HashMapCustom)

Requisitos atendidos:
- Sem uso das estruturas prontas do Java.
- Implementações manuais utilizando arrays e nós encadeados.
- Métodos solicitados no exercício implementados.

## Compilar

```bash
javac src/*.java
```

## Executar

```bash
java -cp src Main
```
